# 開發工具

遊戲開發過程中使用的主要工具如下：

| 工具名稱                        | 用途                                                       |
| ------------------------------- | ---------------------------------------------------------- |
| ChatGPT                         | 劇情撰寫、事件設計與圖像生成協助，提供內容發想與輔助素材。 |
| Python（搭配 Pygame 套件）      | 遊戲主程式開發語言，實作畫面顯示、互動與邏輯流程。         |
| Visual Studio Code（VS Code）   | 開發用程式編輯器，搭配 Git 管理版本與整合模組。            |
| Pixel Studio - pixel art editor | 製作像素風角色與場景的繪圖工具。                           |
