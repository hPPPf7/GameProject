| 選項     | 結果               | 影響   |
| -------- | ------------------ | ------ |
| 認真聽講 | 你獲得了基礎情資。 | fate:1 |

## EV-002 (conditional)

你抵達米瑞村的北門口。天氣晴朗，卻聽見清晰的雨聲。路邊石碑寫著："歡迎回來"。

**條件**: {"step_min": 2, "flag_false": "flag_seen_welcome"}

| 選項             | 結果                                     | 影響                                                                            |
| ---------------- | ---------------------------------------- | ------------------------------------------------------------------------------- |
| 拍照記錄異常情況 | 你留下了照片作為紀錄。                   | fate:1; inventory_add:異常紀錄 - 村口雨聲; flag_set:{'flag_seen_welcome': True} |
| 忽略它，直接進村 | 你決定暫時不理會，繼續前進。             |                                                                                 |
| 觸摸石碑         | 石碑溫熱得像剛被摸過，雨聲彷彿落在腦中。 | fate:2; flag_set:{'flag_welcome_contact': True}                                 |

## EV-003 (conditional)

屋內坐著一名老村民喃喃低語，似乎提及湖邊的石像。你感到莫名熟悉。

**條件**: {"step_min": 3, "flag_false": "flag_heard_watcher_words"}

| 選項                           | 結果                                         | 影響                                                                                |
| ------------------------------ | -------------------------------------------- | ----------------------------------------------------------------------------------- |
| 你在說什麼湖邊？我沒來過這裡。 | 村民低聲回應："那你為何夢見過它？"           | fate:1; flag_set:{'flag_question_identity': True, 'flag_heard_watcher_words': True} |
| 你認錯人了，告訴我你看見什麼。 | 他描述湖邊石像的詭異景象。                   | fate:2; flag_set:{'flag_trust_made': True, 'flag_heard_watcher_words': True}        |
| 吃掉他的聲音                   | 畫面扭曲，村民化為模糊幻影，你陷入短暫失憶。 | flag_set:{'flag_heard_watcher_words': True}                                         |
