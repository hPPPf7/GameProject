# 真結局架構（E3 版）

## 核心概念

主角其實是特殊實驗體，整個「荒野村莊」是一座封閉的模擬測試場。荒謬的事件是用來觀察主角面對未知與道德抉擇時的反應。

## 評價

| 項目         | 評價                                               |
| ------------ | -------------------------------------------------- |
| 世界觀契合度 | ⭐⭐⭐⭐⭐（符合你已有的幻覺、異象、荒謬村民）     |
| 敘事層次感   | ⭐⭐⭐⭐⭐（可做為多重世界線解釋、現實真假模糊）   |
| 玩家驚喜度   | ⭐⭐⭐⭐☆（若設計得當，有很強「啊哈！」反應）      |
| 適合程度     | 極高：能統一所有異常，又解釋主角的「不可信視角」。 |

## 真相內容

- 村民其實是實驗員、演員或 AI，依劇本扮演不同角色。
- 魔物是各種裝置、觀察者或試煉物件。
- 所謂的「調查任務」只是植入記憶的開端。
- 命運值其實代表穩定性指數或自我覺醒程度。

## 主結局版本

| 結局    | 條件                                       | 敘事意涵                   |
| ------- | ------------------------------------------ | -------------------------- |
| A：覺醒 | 發現一切、進入觀察核心、選擇自我銷毀或逃出 | 主角拒絕被操控，結束實驗   |
| B：接受 | 接受自己是實驗體，選擇協助繼續進行實驗     | 成為下一位導師或控制者     |
| C：重置 | 逃避真相或命運值不足，重新啟動模擬場景     | 永遠困在循環裡，回到第一天 |
