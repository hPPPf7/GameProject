# AGENTS.md - 專案與網站同步須知

## 📌 目的 Purpose

本文件提供給所有協作者與 Codex：

> ✅ 每次修改專案內容（如事件、角色、劇情、顯示畫面）後，**請務必同步更新文件網站內容**，以確保使用者與開發者能看到最新版本。

---

## 🧩 本專案的內容與網站關係

- 本專案的網站使用 [MkDocs](https://www.mkdocs.org/) 產生，對應目錄為：`docs/`
- 網站內容由腳本自動根據 `data/` 與其他程式碼生成（位於 `scripts/generate_docs.py`）
- 執行網站更新指令可將最新的遊戲資料、事件、設定等內容反映到網站頁面上

---

## 🔧 哪些變更需要更新網站？

請特別注意，以下情況請同步執行網站更新：

| 類型          | 變更項目                                     |
| ------------- | -------------------------------------------- |
| 資料更新      | 修改 `data/` 資料夾中的 JSON、YAML、文字檔等 |
| 程式邏輯      | 修改 `main.py`、`event_manager.py` 等        |
| UI 或紀錄功能 | 修改 `ui_manager.py`、`text_log.py` 等       |
| 文件生成規則  | 修改 `scripts/generate_docs.py`              |
| 設定或流程    | 修改 `mkdocs.yml` 或 `update_docs.bat`       |
